## Git workflow

```bash
git checkout {branch}
```
Work on branch
```bash
git add <files>
git commit -m 'message'
git push
```
On GitHub
- open Pull Request
- get feedback from the team
- merge code to 'main'

In VSCode
```bash
git checkout main
// Includes changes from branches
git pull
git checkout {branch}
git merge main
```
Continue work
